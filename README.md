# soc_real_time_renderer
A real-time 3D renderer built for a local high school competition, designed as a hands-on exploration of modern rendering techniques. The goal was to implement a broad range of graphics algorithms from scratch and see how they interact in a unified pipeline.

## Features
 - Deferred Rendering
 - Normal Mapping
 - Bloom
 - SSAO
 - TAA
 - Shadow Mapping
 - Terrain Rendering
 - Clouds
 - Atmospheric Scattering
 - Auto Exposure
 - Tone Mapping
